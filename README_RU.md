# Привет!👋
Меня зовут Артур Изланов, я аналитик данных с навыками машинного обучения.

## 👤 О себе

Дата-аналитик с фокусом на повышение эффективности бизнеса через работу с данными. <br>
Мой подход к работе – "зайти, разобраться, оптимизировать":
- Я не просто строю отчеты, а ищу причины, почему процессы буксуют,   
  автоматизирую рутину и перевожу хаос в прозрачные дашборды.
- В работе драйвит результат: сокращение времени на получение данных,   
  рост точности прогнозов и автоматизация процессов, которые раньше отнимали у команды часы. 
- Умею говорить на языке бизнеса: перевожу сложные технические решения   
  в понятные метрики и помогаю стейкхолдерам принимать обоснованные решения.

## 💼 Текущий статус
